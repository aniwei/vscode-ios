# vscode-ios

## How to start?

1、
```
git clone https://github.com/aniwei/vscode-ios.git
cd vscode-ios/Resource/JSRuntime
npm install
npm run build
```

2、
```
git clone https://github.com/aniwei/vscode-ios.git 
yarn 
NODE_ENV=production yarn task build:server:binary
```
